This repository contains my solutions to the exercises from the Lean Introduction textbook by Enric Cosme Llópez and Lü Wong (available at: https://www.uv.es/coslloen/Lean4/).

As you go through the exercises, you’ll notice that the syntax and style improve over time. That’s because I was learning Lean while working through them, so this repo also reflects that progression. Looking back, I’d probably solve the earlier exercises in a cleaner and more idiomatic way now.

All the solutions were written on my own, without using AI, since the goal was really to learn the language properly. After finishing some exercises, I did ask ChatGPT for suggestions on how to make the code more readable. I didn’t directly use its answers, but they helped me understand better practices, which I then tried to apply in later exercises.
